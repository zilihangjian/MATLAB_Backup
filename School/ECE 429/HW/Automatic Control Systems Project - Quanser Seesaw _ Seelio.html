    Automatic Control Systems Project - Quanser Seesaw

    A control system designed in LabVIEW is used to balance a see-saw mechanism using a cart on a track on top. This was a design competition throughout the class, in which our group placed third. This was performed as an end of class project for the Automatic Control Systems class at Purdue University. It was performed with the help of three group members. This class is an example of the root-locus design method using a lead-lag controller. Because this system was not similar to the ideal cases usually discussed in class problems, our group was able to experiment with various design methods and trial and error in order to balance the see-saw.
